# EditableText

`EditableText` component has been built upon InputField component and an added edit button.

## Default variant:

<Canvas>
  <Story name="Default" args={{ label: "Normal text field", fullWidth: true }}>
    {(args) => {
      const [valueText, setValueText] = useState("Hello World");
      return (
        <EditableText
          {...args}
          value={valueText}
          onChange={(e) => setValueText(e.target.value)}
        />
      );
    }}
  </Story>
</Canvas>
