# Modal

`Modal` modal component provides a solid foundation for creating dialogs, popovers,
lightboxes, or whatever else.

## Default variant:

<Canvas>
  <Story
    name="Default"
    args={{ disableEscapeKeyDown: true, disableBackdropClick: true }}
  >
    {(args) => {
      const [open, setOpen] = useState(false);
      return (
        <div>
          <ButtonFilled onClick={() => setOpen(true)}>Open Modal</ButtonFilled>
          <Modal
            {...args}
            open={open}
            onClose={() => setOpen(false)}
            modalActions={
              <ButtonOutlined onClick={() => setOpen(false)}>
                &#x2715;
              </ButtonOutlined>
            }
          >
            <div
              style={{
                padding: "2.5rem",
                fontSize: "2rem",
                marginBottom: "15rem",
              }}
            >
              Modal
            </div>
          </Modal>
        </div>
      );
    }}
  </Story>
</Canvas>
