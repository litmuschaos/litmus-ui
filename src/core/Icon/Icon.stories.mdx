# Icon

Litmus-ui has a set of icons defined to cater to Litmus specifications. Select the `name`
of the icon, its `size` for a list of standard sizes and `color`. Users may also pass `className` to override the css properties
of the SVG element.

<Canvas>
  <Story name="Default" args={{ name: "check" }}>
    {(args) => <Icon {...args} />}
  </Story>
</Canvas>

The size of the icons are mapped to the standard sizes in the following way:

```
xs  : 12 px
sm  : 14 px
md  : 16 px
lg  : 18 px
xl  : 24 px
2xl : 32 px
3xl : 48 px
4xl : 180 px
```
